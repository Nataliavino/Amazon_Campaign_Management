Overview
This repository contains a Python script for managing Amazon Advertising campaigns. The script utilizes the Amazon Advertising API to retrieve campaign data and update campaign statuses based on specified criteria.

Features
Campaign Data Retrieval: Retrieve a list of campaigns associated with a specified profile.
Campaign Status Update: Toggle the status of specified campaigns between "ENABLED" and "PAUSED."
Token Management: Automatically refresh access tokens for secure API authentication.

API URLs and credentials are defined in the config.py file for easy management.
